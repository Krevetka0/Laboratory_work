1. Назовите основные типы, необходимые для работы с GDI+.
Bitmap, Brush, Color, Font, Graphics, Pen, Point, Ellipse, Triangle, Rectangle, Polygon...

2. Какие пространства имен доступны для работы с графикой?
System.Drawing
System.Drawing.Drawing2D
System.Drawing.Imaging
System.Drawing.Printing
System.Drawing.Text

3. Опишите возможные конструкторы класса Point.
Point(Int32), Point(Int32, Int32), Point(Size)

4. В чем состоит особенность конструктора класса Region?
Класс Region представляет внутреннюю часть геометрической фигуры,
поэтому конструкторы класса Region требуют, чтобы им на вход был предоставлен некоторый геометрический объект.

5. Как получить доступ к объекту класса Graphics?
Graphics gr = e.Graphics;
gr.DrawEllipse(...);

6. Опишите способ, с помощью которого можно реализовать перехват события
мыши над фигурами GDI+.
Для начала нужно создать экземпляры класса Region дял фигур, затем с помощью метода Is.Visible() передать координаты мыши по x и y. Если
координаты мыши будут в пределах региона, к которому мы применили данный метод, то будут выполнены те действия, которые мы укажем.

7. Для чего предназначе класс GraphicPath?
Для получения пути к фигуре/линии

8. Чем отличаются типы Point и PointF?
PointF представляет упорядоченную пару координат Х и Y с плавающей запятой, определяющую точку на двумерной плоскости.
Point представляет упорядоченную пару целых чисел — координат Х и Y, определяющую точку на двумерной плоскости.

9. Какие элементы могут обрабатывать событие Paint?
Form, Panel.

10. Для чего используется тип PathPointType?
Для рисования сложных кривых линий

11. Через какой тип происходит рисование в обработчике события Paint?
класс Graphics

12. Опишите механизм выбора объекта GDI.
Для выбора любого объекта можно использовать функцию SelectObject.

13. Какие события участвуют в процессе интрактивного взаимодействия с
GDI объектами графики?
MouseMove, MouseDown, Click, DoubleClick

14. Какое событие необходимо задействовать, если добавить в программу
отображение контура объекта при перетаскивании?
Paint, IsImageClicked

15. Опишите механизм реализации анимации с использованием GDI+.
После изменение чего-то на форме (или другом объекте, на котором происходит анимация), её нужно перерисовать с помощью метода 
Invalidate().

16. Опишите назначение основных свойств и методов класса Timer.
Свойство Interval определяет временный промежуток одного тика таймера; методы Start() и Stop() запускают и останавливают таймер 
соответственно.

17. Для чего предназначено событие Tick класса Timer?
Событие Tick предназначено для того, чтобы отображать, когда указанный интервал таймера заканчивается.

18. Сколько таймеров может присутствовать в программе?
Очень много, давайте мы будем тыкать таймеры на форму пока комп не сгорит!
